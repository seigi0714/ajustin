# adjustin_app
空いてる時間予定をアジャスト。
面倒な日程調整はこのアプリにお任せ

## 機能、要件

### リリース時必須

- 自分の空いた時間、場所などを指定してプランを立てることができる
- URLを送ればその人をプランに招待できる
- 空き時間に設定しているところはリアルタイムで更新できる
- googleカレンダーと連携
- 

### 実装予定

- プランを共有する時広告画像みたいなのを作って共有できる
- ゆるぼが出来る。
